# AI Typing Assistant 🎹

A web-based typing assistant designed specifically for users with typing impairments or disabilities. Uses AI to provide real-time spelling, grammar, and predictive text suggestions.

## Features ✨

- **Real-time spelling corrections** - Fixes common typos automatically
- **Grammar suggestions** - Improves sentence structure
- **Predictive text** - Suggests next words based on context
- **Auto-correct mode** - Automatic corrections as you type
- **One-click fixes** - Easy correction suggestions
- **Mobile responsive** - Works on all devices

## Quick Start 🚀

1. **Browse Website**: Simply open `index.html` in any web browser
2. **No installation required** - Fully client-side JavaScript
3. **Start typing** - The AI assistant will provide real-time suggestions

## Demo Use Cases 🎯

Perfect for:
- **users with dyslexia** - Catches dyslexic typing patterns
- **users with hand-issues** - Corrects mis-clicks and delays
- **English learners** - Grammar and spelling guidance
- **Senior users** - Large fonts and simple interface

## How It Works 🔧

1. **Real-time processing** - Analyzes text as you type
2. **AI suggestions** - Uses frequency-based word prediction
3. **Context awareness** - Understands sentence structure
4. **One-click fixes** - Easy correction interface
5. **Privacy first** - No data sent to external servers

## Tech Stack 🛠️

- **Frontend**: Pure HTML5, CSS3, ES6 JavaScript
- **AI Engine**: Client-side predictive algorithms
- **Responsive**: Mobile-first design
- **Accessibility**: ARIA labels and keyboard navigation

## Customization 🎨

### Easy modifications:
- Add custom words to dictionary
- Adjust spelling rules
- Change theme colors
- Modify suggestion sensitivity

## Usage Examples 📋

**For users with hand tremors:**
- Corrects "wiht" → "with"
- Fixes "thsi" → "this"
- Handles extra spaces

**For dyslexic users:**
- Fixes transposed letters: "hte" → "the"
- Corrects similar-sounding words: "recieve" → "receive"

**For English learners:**
- Shows "dont" → "don't" (apostrophe correction)
- Suggests better word choices

## Bookmarklet Integration 📱

Can be easily embedded as a bookmarklet to work on any website.
